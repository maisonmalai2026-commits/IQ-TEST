# CogniScale — Professional IQ-Style Practice Test

This is a static website you can upload for free to GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any normal web host.

## Files

- `index.html` — main webpage
- `styles.css` — responsive professional styling
- `app.js` — timer, questions, scoring, explanations, local best score

## Important wording

The site uses truthful wording: **Professional IQ-Style Practice Test**. It does not claim to be a clinical assessment or official IQ score.

## How to upload free

### GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and `app.js` to the repository root.
3. Go to Settings → Pages.
4. Choose branch `main` and root folder.
5. Save. GitHub gives you a public website URL.

### Netlify
1. Go to Netlify and choose “Add new site”.
2. Drag the whole folder into the deploy box.
3. Netlify gives you a public website URL.
